RTS Map App

This is an RTS Simulator project that I made in my java 2 class in Spring 2013. The simulator represents bus stops, points of interests, and bus lines. Each bus line consists of a sequence of bus stops, which is the route. Some of the bus stops on a route are called time points, at which the bus is supposed to be at a specific point in time as mentioned in the schedule. In other words, the schedule only lists the time point type of bus stops. This application contains many features such as a visual representation of bus lines, where the user can draw a thick line on the map to represent the bus line graphically. There is also a query feature. The user can search for the best Line when he/she wants to go from a point of Interest (departure) to another point of Interest (destination) on the map. Basically, the application will search for the line that stops at a bus stop (departure bus stop) closest from the departure point of interest and goes to a bus stop closest to the destination point of interest.




Rat In Maze

This is the typical "Rat In Maze" project that I coded for my data structures class in Fall 2013. The "Rat in maze problem" consists of finding a path in a maze, with given starting and ending points. The maze is given as a 2D array, where an open path is a sequence of positions represented by '0's. The rat(s) cannot move to walls which are represented by '1's. For every position "taken" by the rat, the position changes from '0' to '2'. The coordinates of the square at the top left corner are (0,0). The coordinates of the square at the bottom right corner are (12,14).
