Research

I have done work on software reliability. I worked on a project to automate the discovery of configuration parameters in Hadoop that might impact certain behavior. I used a static program analysis technique called Thin Slicing to extract relevant configuration parameters. Ultimately, the goal of the project was to provide suggestions on how to tune the relevant configuration parameters based on specific source-code level metrics. Another project I worked on was to create a tool that detects changes in code that may introduce potential deadlocks. I used WALA as the static analysis framework for both of these projects.
